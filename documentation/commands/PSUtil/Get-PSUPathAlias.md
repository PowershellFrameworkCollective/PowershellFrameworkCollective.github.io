# Get-PSUPathAlias

## SYNOPSIS
Gets the PSUPathAlias configuration values.

## SYNTAX

```
Get-PSUPathAlias [[-Alias] <String>] [<CommonParameters>]
```

## DESCRIPTION
Gets the PSUPathAlias configuration values from the PSFConfig system.

## EXAMPLES

### EXAMPLE 1
```
Get-PSUPathAlias
```

Returns all aliases

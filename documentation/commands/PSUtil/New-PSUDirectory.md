# New-PSUDirectory

## SYNOPSIS
Creates a folder and moves the current path to it.

## SYNTAX

```
New-PSUDirectory [-Path] <Object> [<CommonParameters>]
```

## DESCRIPTION
Creates a folder and moves the current path to it.

## EXAMPLES

### EXAMPLE 1
```
mcd Test
```

creates folder C:\Test, then moves the current location to it.

## PARAMETERS

### -Path
Name of the folder to create and move to.

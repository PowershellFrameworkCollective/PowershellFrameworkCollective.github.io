# Invoke-MDDaemon

## SYNOPSIS
Processes the email queue and sends emails

## SYNTAX

```
Invoke-MDDaemon [<CommonParameters>]
```

## DESCRIPTION
Processes the email queue and sends emails.
Should be scheduled using a scheduled task.
Recommended Setting:
- Launch on boot with delay
- Launch on Midnight
- Repeat every 30 minutes for one day

## EXAMPLES

### EXAMPLE 1
```
Invoke-MDDaemon
```

Processes the email queue and sends emails
